03Bước kế tiếp — hai hướng
Hướng A
Đưa lên production

Biến sản phẩm chạy local thành online thật.

  • web → Vercel hoặc Cloudflare Pages (SSR/ISR sẵn)
  • core → VPS / Coolify (Docker + env thật)
  • Postgres managed hoặc VPS (backup)
  • R2 Custom Domain → CDN cache/WAF thật thay r2.dev
Hướng B
Phase 2 — AI

Tận dụng pgvector + outbox đã sẵn trong stack.

  • Chatbot RAG hỏi đáp nội dung blog (embedding + pgvector)
  • AI worker Python + LangGraph, consume outbox
  • Research tự viết bài duyệt qua Telegram (human-in-the-loop)
  • Trợ lý viết gợi ý ngay trong editor

Chưa chốt hướng — bàn sau. Cả hai độc lập; có thể deploy trước rồi mới AI.
